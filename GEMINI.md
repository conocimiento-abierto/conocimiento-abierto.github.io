# Project Overview

This project is a static, multilingual website that displays the "Decalogue of Open Knowledge." It allows users to switch between several languages (Spanish, Catalan, Galician, Basque, and English) to read the ten principles of open knowledge. The website is designed to be simple, responsive, and printer-friendly.

## Key Files

*   `index.html`: The main HTML file that contains the structure of the website. It includes the text for the decalogue in all available languages, with a language selector to toggle between them.
*   `style.css`: The stylesheet for the project. It defines the visual appearance of the page, including layout, colors, fonts, and responsive design for different screen sizes. It also includes specific styles for printing.
*   `script.js`: This JavaScript file provides the interactivity for the language selector. It handles showing and hiding the content for the selected language and saves the user's language preference in `localStorage`. It also includes the functionality for the print button.
*   `abierto.png` / `abierto_reducido.png`: Image files used as the logo for the project.

## Usage

This is a static website. To view it, simply open the `index.html` file in a web browser. There is no build process or server-side logic required. The project is self-contained and can be hosted on any static web hosting service.
